## рҹҸ—пёҸ Principios de Arquitectura Aplicados

### рҹҺҜ Single Responsibility Principle (SRP)
- **Routes**: Solo manejan HTTP requests/responses
- **Controllers**: Solo lГіgica de negocio
- **Models**: Solo acceso a datos
- **Templates**: Solo presentaciГіn

### рҹ”„ Dependency Inversion Principle (DIP)
- **Controllers** dependen de abstracciones (Models)
- **Routes** dependen de abstracciones (Controllers)
- **FГЎcil testing** y mantenimiento

### рҹ“Ұ Separation of Concerns
- **PresentaciГіn**: Templates + CSS + JS
- **LГіgica**: Controllers
- **Datos**: Models + Database
- **ConfiguraciГіn**: Config files
