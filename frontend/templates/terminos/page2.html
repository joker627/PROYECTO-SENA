TÉRMINOS Y CONDICIONES DE USO

¡Gracias por usar Sign Technology!

1Aceptación de Términos

Al registrarse y utilizar el sistema de traducción de señas de Sign Technology, el usuario acepta de manera voluntaria y consciente estos Términos y Condiciones, así como la Política de Privacidad asociada.

2Propósito del Software

Sign Technology es un sistema diseñado para facilitar la comunicación mediante la traducción de lenguaje de señas, proporcionando una herramienta tecnológica innovadora para mejorar la accesibilidad.

3Consentimiento para el uso de datos anonimizados

El usuario autoriza a Sign Technology a almacenar y utilizar de manera anónima los datos de movimiento (landmarks) generados durante el uso del sistema. Estos datos se utilizarán exclusivamente para el perfeccionamiento del software, incluyendo procesos de mejora, ajuste de errores y entrenamiento del sistema.

4¿Qué son los landmarks?

Los landmarks son representaciones numéricas generadas por el sistema que identifican puntos clave del cuerpo, rostro y manos capturados mediante cámara. No son fotografías ni grabaciones, y no contienen información personal o biométrica reconocible. Su única finalidad es técnica: servir como base para interpretar los movimientos y señas realizados por el usuario.

5Almacenamiento obligatorio de datos

El almacenamiento de los landmarks es una condición esencial para el funcionamiento del sistema. No es posible utilizar el software sin autorizar este proceso.

Los datos se registran de manera totalmente anónima, sin asociarse al nombre del usuario ni a ningún dato que permita su identificación directa.

6Eliminación de cuentas y conservación de datos

El usuario puede solicitar la eliminación de su cuenta y sus datos personales del sistema en cualquier momento. Sin embargo, los datos de landmarks generados durante el uso del software no serán eliminados, ya que forman parte de un registro anónimo utilizado para mejorar la herramienta y garantizar su funcionamiento a largo plazo.

7Modificaciones y actualizaciones

Sign Technology podrá modificar estos Términos y Condiciones según lo requiera la evolución del proyecto. En caso de cambios sustanciales, se informará al usuario dentro de la aplicación antes de su aplicación.

Confirmación de consentimiento

Durante el proceso de registro, el usuario deberá aceptar expresamente los Términos y Condiciones marcando la siguiente casilla:

"He leído y acepto los Términos y Condiciones de uso, incluyendo el uso obligatorio de mis datos de movimiento (landmarks) de forma anónima para fines de mejora del sistema."
